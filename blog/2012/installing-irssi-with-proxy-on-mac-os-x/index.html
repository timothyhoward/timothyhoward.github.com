Installing IRSSI with Proxy on Mac OS X

I couldn’t find any relevant documentation in regards to installing irssi on Mac OS X, so I thought I’d post some instructions for future reference and to assist any of those out there that wish to try irssi. For those that don’t know, irssi is a powerful text-based IRC client that can be run on Unix-like systems which include the likes of Mac OS X and its BSD counterparts, as well as distributions powered by the Linux kernel. The most useful feature - in my opinion - is the irssi-proxy function which allows for irssi to be run on a server inside GNU Screen (to act as a daemon and maintain a consistent IRC connection), acting as a bouncer/proxy for other clients to connect. In other words, say you had a iPhone or another Mac with an IRC client; you could configure these clients to connect to the proxy, and it would act as one consistent IRC session. If you were to disconnect either of the clients, the server providing the irssi-proxy would continue to be connected to IRC. Plugins can also be installed to even allow for backlogs, making this setup extremely powerful and useful. But enough of talking about it, let’s install it!

Installing Homebrew

If you don’t already have Homebrew, now is the time to get it. Homebrew is a brilliant package manager for OS X, similar to the likes of apt-get and pacman. Once installed, Homebrew uses ‘formulas’, which are basic Ruby scripts that hold the commands to configure, patch, compile and install packages on to OS X. It figures out the quirks of OS X for you, so you don’t have to! To install, simply type:

/usr/bin/ruby -e "$(curl -fsSL https://raw.github.com/gist/323731)"

This will install Homebrew into /usr/local so that you won’t have to sudo when you brew install. Now, onwards to installing the dependencies of IRSSI.

Prerequisites

There are a few things which will need to be installed before IRSSI can be compiled and installed itself. These packages are:

  • Glib - A core suite of libraries for Unix-like applications written in C.
  • pkg-config - A program used to retrieve information about installed libraries.
  • Gettext - A set of tools for multi-language support in GNU applications.
  • Libiconv - A text encoding converter, used to convert between various character encodings in international text.
  • Libffi - A portable, high level programming interface to various calling conventions.

To install these dependencies, type these commands in order:

brew install pkg-config
brew install glib
brew install libiconv

Please note that gettext and libffi are installed with glib. Alright, almost done!

Installing IRSSI

Homebrew is so good that we can even install IRSSI with it! In this tutorial, I am going to install perl support and irssi-proxy, as well as irssi itself. To install, type:

brew install irssi --with-perl=yes --with-proxy

Let the process run, and it will end with a symlink error:

==> make install
ln: module.h: Permission denied
Error: The linking step did not complete successfully
The formula built, but is not symlinked into /usr/local
You can try again using `brew link irssi'

This error is the result of a mistake in the Homebrew formula for irssi; it’s simply missing sudo! To rectify, type:

sudo brew link irssi

Type your password into the prompt, and you’re good to go! Further configuration documentation can be found on the irssi site, or the excellent documentation on Quadpoint detailing how to run irssi in GNU Screen as previously mentioned. If you have trouble with anything, leave a comment below and I’ll try to assist!
